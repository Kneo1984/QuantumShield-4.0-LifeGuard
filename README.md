## 🚀 Projektüberblick

**QuantumShield™ 4.0 - LifeGuard** ist ein vollständig autonomes, lokal gesteuertes Schutzsystem.

- 🔒 **Auto-VPN-Start** (WireGuard Tunnelmanagement)
